About Friendly Links

Apr 28th, 2014

In fact, I did not SEO my website carefully, I moved my blog from wordpress to octopress last year.

My blog include so many catagories, like Octoprss, Linux , Religion, Blender and So on , that it is not good to ger higher pagerank. And I even did not write the articles with headings, keywords or Bold tags in most of time.

My purpose to build <baham.co> is just to write some articles for myself and for the whole Internet.

But recently, I found the PageRank value of my website get 2.

So I decided add a sidebar of Friendly Links

I am happy to exchange links to everyone who has a website that the main content is similar with my website.

You know the scope of my blog’s content is very wide. So most websites can add the friendly links. ^_^

Can contact me! to talk about add a friendly link.

Notice

The bumber of friendly links in home page will not be more than 3.

So when there are already 3 links in home page, if you still want to exchange links, I can only add the link to a friendly links page.
